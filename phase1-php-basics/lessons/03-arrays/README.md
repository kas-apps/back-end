## 🎯 なぜ配列を学ぶの？（Why）

### データの「入れ物」が手に入る！

想像してみて！商品を1つずつ変数に入れるとこうなる：

```php
<?php
$product1 = "りんご";
$product2 = "バナナ";
$product3 = "オレンジ";
// ...100個あったら大変！
?>
```

**配列を使えば**：

```php
<?php
$products = ["りんご", "バナナ", "オレンジ"];
// すっきり！
?>
```

### Excelの「列」みたいなもの！

配列は、Excelの列に似ているよ！

| 番号 | 商品名 |
|-----|--------|
| 0   | りんご |
| 1   | バナナ |
| 2   | オレンジ |

PHPの配列：
```php
$products[0] = "りんご";
$products[1] = "バナナ";
$products[2] = "オレンジ";
```

## 💾 配列の基本（What）

### 添え字配列

**基本的な配列**：

```php
<?php
// 配列の作成
$colors = ["赤", "青", "緑"];

// 要素の取得（0から始まる）
echo $colors[0];  // 赤
echo $colors[1];  // 青
echo $colors[2];  // 緑
?>
```

**配列への要素追加**：

```php
<?php
$fruits = ["りんご", "バナナ"];

// 末尾に追加
$fruits[] = "オレンジ";

// または
array_push($fruits, "ぶどう");

// 結果: ["りんご", "バナナ", "オレンジ", "ぶどう"]
?>
```

### 連想配列

**キーと値のペア**：

```php
<?php
// ユーザー情報を連想配列で管理
$user = [
    "name" => "山田太郎",
    "age" => 25,
    "email" => "taro@example.com"
];

// 値の取得
echo $user["name"];   // 山田太郎
echo $user["age"];    // 25
?>
```

### 配列操作関数

```php
<?php
$numbers = [1, 2, 3, 4, 5];

// 要素数を取得
echo count($numbers);  // 5

// 要素が含まれているか確認
if (in_array(3, $numbers)) {
    echo "3が含まれています";
}

// 配列の結合
$merged = array_merge([1, 2], [3, 4]);  // [1, 2, 3, 4]
?>
```

### foreachループ

```php
<?php
$fruits = ["りんご", "バナナ", "オレンジ"];

// 配列の各要素を処理
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

// 連想配列の場合
$user = ["name" => "太郎", "age" => 25];

foreach ($user as $key => $value) {
    echo $key . ": " . $value . "<br>";
}
?>
```
