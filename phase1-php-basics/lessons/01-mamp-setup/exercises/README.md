# Lesson 01: MAMP環境のセットアップ - 演習問題 💪

実際に手を動かして、MAMP環境に慣れよう！

---

## 📝 基礎編

### 問題1：自分の名前を表示しよう

**課題**：
`myname.php` というファイルを作成し、あなたの名前を表示してください。

**期待される出力**：
```
私の名前は山田太郎です。
```

**ヒント**：
- `<?php ?>` でPHPコードを囲む
- `echo` を使って文字列を出力
- セミコロン `;` を忘れずに

**ファイルの場所**：
```
htdocs/myname.php
```

**アクセスURL**：
```
http://localhost:8888/myname.php
```

---

### 問題2：現在の日時を表示しよう

**課題**：
`today.php` というファイルを作成し、現在の日時を表示してください。

**期待される出力**：
```
今日は2025年11月17日です。
現在の時刻は14:30:00です。
```

**ヒント**：
- `date()` 関数を使う
- `date('Y年m月d日')` で日付を取得
- `date('H:i:s')` で時刻を取得

**ファイルの場所**：
```
htdocs/today.php
```

---

### 問題3：PHPのバージョンを確認しよう

**課題**：
`version.php` というファイルを作成し、PHPのバージョンを表示してください。

**期待される出力**：
```
PHPのバージョン：8.0.30
```

**ヒント**：
- `phpversion()` 関数を使う
- `echo` と文字列連結（`.`）を使う

**ファイルの場所**：
```
htdocs/version.php
```

## 🎯 応用編

### 問題4：複数の情報を表示しよう

**課題**：
`info.php` というファイルを作成し、以下の情報を表示してください：

1. あなたの名前
2. 今日の日付
3. PHPのバージョン
4. Webサーバー（Apache）の情報

**期待される出力**：
```
=== システム情報 ===
名前：山田太郎
日付：2025年11月17日
PHPバージョン：8.0.30
Webサーバー：Apache
```

**ヒント**：
- `$_SERVER['SERVER_SOFTWARE']` でWebサーバー情報を取得
- 複数の `echo` を使うか、改行（`<br>`）を使う
- HTMLタグも使える！

**ボーナス**：
HTMLを使って、見た目を整えてみよう！

---

### 問題5：エラーを修正しよう

**課題**：
以下のコードにはエラーがあります。修正して、正しく動くようにしてください。

**間違ったコード**：
```php
<?php
echo "こんにちは"
echo "世界！";
```

**期待される出力**：
```
こんにちは
世界！
```

**問い**：
1. どこが間違っているか見つけてください
2. 修正したコードを `fixed.php` として保存してください
3. ブラウザで確認してください

**ヒント**：
- セミコロンがどこかに足りない？
- エラーメッセージをよく読もう

## 🌟 チャレンジ

### 問題6：自己紹介ページを作ろう

**課題**：
`profile.php` というファイルを作成し、あなたの自己紹介ページを作ってください。

**含めるべき情報**：
- 名前
- 年齢または誕生年
- 好きなもの（3つ）
- 今日の日付と時刻
- PHPを学んでいることのメッセージ

**期待される出力例**：
```
===================
自己紹介
===================

名前：山田太郎
年齢：25歳

好きなもの：
・プログラミング
・音楽
・旅行

今日は2025年11月17日 14:30:00です。
私はPHPを学んでいます！

===================
```

**条件**：
- HTMLタグを使って見やすくする（`<h1>`、`<p>`、`<ul>`など）
- CSSでスタイルを付ける（オプション）
- 変数を使う（`$name`、`$age` など）

**ボーナスチャレンジ**：
- 背景色を変える
- フォントを変える
- 画像を追加する

**課題**：
AIに以下の指示を出して、コードを生成してもらってください。

**指示**：

## 🆘 困ったときは

### エラーが出たら

1. **エラーメッセージを読む**
   - 何行目でエラーが起きているか確認
   - エラーの種類を確認

2. **よくあるミスをチェック**
   - セミコロン `;` を忘れていないか
   - `<?php ?>` で囲んでいるか
   - クォート `"` や `'` が閉じているか

3. **AIに質問する**
   - エラーメッセージをそのまま見せる
   - コードも一緒に見せる

### 表示がおかしいとき

1. **ファイルの拡張子を確認**
   - `.php` になっているか

2. **MAMPが起動しているか確認**
   - MAMPウィンドウでライトが緑色になっているか

3. **URLを確認**
   - `http://localhost:8888/ファイル名.php` で正しいか
