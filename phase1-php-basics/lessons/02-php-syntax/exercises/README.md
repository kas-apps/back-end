### 問題10：関数の分割

**課題**：
以下のコードを、適切な関数に分割してください。

**分割前のコード**：
```php
<?php

$discount = 0.2;
$discountedPrice = $price - ($price * $discount);
$taxRate = 0.1;
$finalPrice = $discountedPrice + ($discountedPrice * $taxRate);
echo "最終価格：" . $finalPrice . "円";
?>
```

**改善ポイント**：
1. 割引計算を関数にする（`applyDiscount`）
2. 税込計算を関数にする（`addTax`）
3. コメントを追加する

**ファイル名**：`02-10-function-split.php`

### 問題13：簡易BMI計算機

**課題**：
身長（cm）と体重（kg）から BMI を計算し、判定を表示するプログラムを作成してください。

**BMIの計算式**：
```
BMI = 体重(kg) ÷ (身長(m) × 身長(m))
```

**判定基準**：
- 18.5未満：やせ
- 18.5〜25未満：標準
- 25〜30未満：肥満（軽度）
- 30以上：肥満（重度）

**期待される出力**：
```
身長：170cm
体重：65kg
BMI：22.5
判定：標準
```

**ヒント**：
- 身長をcmからmに変換（`$height / 100`）
- `round($bmi, 1)` で小数点第1位まで表示

**ファイル名**：`02-13-bmi-calculator.php`

## 🆘 困ったときは

### エラーが出たら

1. **Parse error**：構文エラー
   - セミコロン `;` を忘れていないか
   - `{` と `}` の数が合っているか
   - クォート `"` や `'` が閉じているか

2. **Undefined variable**：変数が定義されていない
   - 変数名のスペルミス
   - スコープの問題（関数の内外）

3. **Call to undefined function**：関数が定義されていない
   - 関数名のスペルミス
   - 関数を定義する前に呼び出していないか

### AIへの質問例

```text
「このエラーが出ました：
[エラーメッセージ]

コードはこれです：
[コード]

どこが間違っているか教えてください」
```
