# Mini Calculator - Phase 1 総合プロジェクト 🧮

**目的**：Phase 1で学んだすべての知識を統合して、セキュアな電卓アプリを作成する！

---

## 🎯 プロジェクト概要

このプロジェクトでは、Webベースの簡易電卓を作成します。
Phase 1で学んだ以下の知識をすべて使います：

- 変数とデータ型
- 条件分岐
- 関数
- フォームデータの受け取り
- バリデーション
- **XSS対策**

---

## ✨ 機能

- 2つの数値と演算子（+、-、×、÷）を入力
- 計算結果を表示
- **バリデーション**：
  - 数値チェック
  - ゼロ除算チェック
  - 入力必須チェック
- **XSS対策済み**
- エラーメッセージの表示

---

## 📁 ファイル構成

```text
mini-calculator/
├── README.md          # このファイル
├── index.php          # メインファイル（フォームと結果表示）
└── calculator.php     # 計算ロジック（関数）
```

---

## 💻 使い方

1. MAMPを起動
2. `htdocs/mini-calculator/` にファイルを配置
3. ブラウザで `http://localhost:8888/mini-calculator/` にアクセス
4. 数値を入力して計算！

---

## 🔍 実装のポイント

### セキュリティ

✅ **XSS対策**：すべての出力で `htmlspecialchars()` を使用
✅ **バリデーション**：ユーザー入力を信用しない
✅ **エラーハンドリング**：適切なエラーメッセージ

### コード品質

✅ **関数化**：計算ロジックを関数に分離
✅ **コメント**：重要な処理に説明
✅ **わかりやすい変数名**
