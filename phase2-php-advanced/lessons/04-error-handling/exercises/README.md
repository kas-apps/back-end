### 演習 04-03: 情報漏洩の修正

**課題**：以下の脆弱なコードを修正しよう。

```php
<?php
// 🚨 本番環境で詳細なエラーを表示（情報漏洩）
$file = file_get_contents("/path/to/secret.txt");
?>
```

**修正要件**：

- エラーを表示しない
- ユーザーには一般的なメッセージ
- 詳細はログに記録
