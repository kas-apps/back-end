# Lesson 04: セキュリティ対策 🔒

**学習目標**：Webアプリケーションの主要なセキュリティ脆弱性を理解し、すべての対策を実装できるようになる！

## 5つのセキュリティ対策

1. **SQLインジェクション対策** → プリペアドステートメント
2. **XSS対策** → htmlspecialchars()
3. **CSRF対策** → トークン
4. **パスワード保護** → password_hash()
5. **セッション管理** → session_regenerate_id()

すべて実装して、セキュアなWebアプリケーションを作ろう！
